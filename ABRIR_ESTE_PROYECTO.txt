INSTRUCCIONES IMPORTANTES PARA ABRIR EL PROYECTO

1. Abre en IntelliJ la carpeta: quindioflix-backend
   No abras la carpeta target/classes.

2. El código fuente editable está en:
   src/main/java/co/edu/uniquindio/quindioflix

3. Si IntelliJ muestra "Decompiled .class file", significa que abriste una clase compilada de target/ o una dependencia Maven.
   Eso no es el código fuente del proyecto. El proyecto entregado no incluye target/.

4. Para ejecutar en modo desarrollo, usa:
   mvn clean spring-boot:run

5. Para Swagger:
   http://localhost:8080/swagger-ui.html

6. Usuario de prueba:
   email: usuario1@mail.com
   password: Password123

7. Para Oracle/SQL Developer:
   - Ejecuta database/00_crear_usuario_app.sql con un usuario administrador.
   - Ajusta application-oracle.yml si tu PDB no se llama FREEPDB1.
   - Ejecuta: mvn spring-boot:run -Dspring-boot.run.profiles=oracle
