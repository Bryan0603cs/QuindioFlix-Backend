IMPORTANTE PARA ABRIR EN INTELLIJ

1. Abre la carpeta quindioflix-backend, no abras target/classes.
2. El código fuente editable está en:
   src/main/java/co/edu/uniquindio/quindioflix
3. Si IntelliJ muestra "Decompiled .class file, bytecode version 61.0 (Java 17)", estás viendo una clase compilada o una dependencia. Eso no es un error.
4. Para probar sin Oracle usa el perfil dev:
   mvn clean spring-boot:run
5. Para probar con Oracle:
   mvn clean spring-boot:run -Dspring-boot.run.profiles=oracle
6. En el perfil dev se desactivó la validación de esquema de Hibernate para evitar el error de H2 con NUMBER/DECFLOAT.
7. En Oracle se mantiene ddl-auto=validate para validar el modelo real.
